# 데카르트 곱 (Cartesian Product)

두 집합의 **모든 조합을 쌍으로 만든 것**입니다.

---

### 예시

```
A = {1, 2}
B = {a, b, c}

A × B = {
  (1, a), (1, b), (1, c),
  (2, a), (2, b), (2, c)
}
```

A의 **각 원소**와 B의 **모든 원소**를 빠짐없이 짝지은 것.

결과 크기 = `|A| × |B|` = 2 × 3 = **6개**

---

### 개발에서 만나는 곳

**SQL JOIN (조건 없을 때)**

```sql
SELECT * FROM A, B  -- WHERE 없으면 데카르트 곱 발생
```

A가 100행, B가 100행이면 → 10,000행 반환. 실수로 날리면 DB 터짐.

**중첩 반복문**

```java
for (String color : colors) {
    for (String size : sizes) {
        // color × size 의 모든 조합
    }
}
```

옷 상품의 색상 × 사이즈 옵션 생성 같은 것.

---

### 한 줄 요약

> "A의 모든 것" × "B의 모든 것" = **가능한 모든 조합**
