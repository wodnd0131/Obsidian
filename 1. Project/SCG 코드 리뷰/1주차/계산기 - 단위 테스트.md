# 🚀 1단계 - 초간단 계산기 구현
## 기능 요구사항
- 인자 2개를 받아 사칙연산을 할 수 있는 계산기를 구현한다.
- 사칙연산과 매칭되는 4개의 메서드를 제공한다.
- 계산된 결과는 정수를 반환한다.

## 새로운 프로그래밍 요구사항
- **메인 메서드는 만들지 않는다.**
# 🚀 2단계 - 초간단 문자열 테스트
## 새로운 프로그래밍 요구사항
- 구현한 초간단 계산기가 예상한대로 동작하는지 Junit5를 활용하여 테스트를 자동화한다.

## 기존 프로그래밍 요구사항

- 메인 메서드 없이 동작을 검증하는 경험을 해본다.
- 메인 메서드는 만들지 않는다.
