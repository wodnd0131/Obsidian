** 성능 테스트 환경 OOM 문제
성능 테스트 서버에서 발생하는 OOM문제는 템플릿을 조회할때 100만건의 템플릿이 모두 조회되기 때문이었습니다!
이를 방지하기 위해 예전에 PR에 템플릿 조회 API에 페이징을 적용해놓은 상태입니다.
PR만 머지되면 해결될 예정입니다!


로컬 성능 테스트 환경을 구축한 내용을 노션으로 정리해놓았습니다.
현재 배포된 모니터링 서버에 있는 DB 대시보드는 미완성이었는데 이번에 로컬 성능 테스트 환경에는 완성해서 적용해놓았습니다!
