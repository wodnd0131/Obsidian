# java.lang.Character
### 1. `Character.isDigit()` 이란

```java
Character.isDigit('3')   // true
Character.isDigit('③')   // true  ← 여기서 문제
Character.isDigit('٣')   // true  ← 아라비아 숫자
Character.isDigit('Ⅲ')   // false (로마숫자는 해당 안됨)
```

Java의 `Character.isDigit()`은 **유니코드 기준**으로 "숫자"를 판단합니다. 우리가 생각하는 `0~9` 만이 아니에요.

---

### 2. 유니코드 DECIMAL_DIGIT_NUMBER

유니코드는 전 세계 모든 문자를 다루는 표준입니다. 그 안에서 "십진수 숫자"로 분류된 카테고리가 `DECIMAL_DIGIT_NUMBER`이고, 여기엔 각국의 숫자 표기가 다 들어갑니다.

```
0-9       (ASCII 숫자)
٠-٩       (아라비아-인도 숫자)
০-৯       (벵골 숫자)
๐-๙       (태국 숫자)
③ ④ ⑤    ... 등
```

`Character.isDigit()`은 이걸 **전부 true**로 반환합니다.
