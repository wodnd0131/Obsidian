# [워크 플로우](https://app.riido.io/CtSCeysiKd1I5R3-v3LMt/teams/SA/list)
![[../../Repository/image (3).png]]
위에서부터 “프로젝트 - 목표 - 작업”의 계층 구조로 이루어진 워크 플로우

### 1. 프로젝트

주차별 목표, SWYP 일정 반영 매주 `고정 회의` 때 작업 분배 진행

-> 회의를 통하여 목표 선정, 해당 내용을 기록 중요한 것은 목표마다의 마감일 설정이 진행될 것

### 2. 목표

각 파트별 금주 목표. `파트 회의`에서 세부 사항 결정을 진행한다.

-> 파트 팀장과 함께 하위 작업 분배, 해당 내용 토의 내용을 함께 기록

중요한 것은 작업마다의 마감일, 노력추정치(예상 작업 시간) 설정이 진행될 것. 또한, 이때 중요도도 함께 설정되어야 한다.

### 3. 작업

파트 목표에 따른 실제 세부 작업 담당자가 내용을 작성 및 진행 상태 관리.

+) 이때, 뤼이도 유료 기능 활용하여, 우측 상단 AI로 본문 작성, 등록일 자동 매칭 등 가능하니 참고
