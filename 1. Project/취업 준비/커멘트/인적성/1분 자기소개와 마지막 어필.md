### 1분 자기소개

> 안녕하십니까, Yapp 29기  서버 파트에 지원한 백엔드 개발자 황재웅입니다.
> 
> 저는 팀의 병목을, 재사용할 수 있는 구조로 해결하는 것에 집중해왔습니다.
> 
> 지난 프로젝트에서 도메인 구조 개편으로 인해 프론트엔드 작업이 반복적으로 멈추는 걸 발견했습니다.
> 저는 API 버저닝을 도입해 양쪽이 독립적으로 개발할 수 있는 환경을 만들었고, 
> 협업 대기 시간을 단축했습니다.
> 
> 또한, 배포 실패 시 발생하는 5분 내외의 서비스 중지 시간을 해결하고자 자동 롤백 시스템을 설계했습니다. Docker 리태깅 전략과 타임스탬프 기반 DB 백업을 활용해 배포 사고 시 즉각적으로 이전 버전으로 전환할 수 있는 구조를 만들었고, 그 결과 복구 소요 시간을 기존 5분에서 30초 이내로 단축했습니다.
> 
> YAPP에서는 지금까지 해온 것처럼, 팀이 더 빠르게 움직일 수 있는 구조를 만들어가고 싶습니다.
