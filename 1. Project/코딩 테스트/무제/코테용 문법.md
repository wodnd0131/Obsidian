## Arrays / Collections 유틸

|명령어|용도|
|---|---|
|`Arrays.sort(arr)`|배열 정렬|
|`Arrays.sort(arr, Comparator.reverseOrder())`|Integer[] 역순 정렬 (int[]는 불가)|
|`Arrays.stream(arr)`|배열 → Stream 변환|
|`Arrays.asList(arr)`|배열 → List 변환 (고정크기, 수정 불가)|
|`Arrays.fill(arr, val)`|배열 전체를 val로 채움|
|`Arrays.copyOf(arr, len)`|배열 복사/자르기|
|`Collections.sort(list)`|List 정렬|
|`Collections.max(list)` / `min(list)`|최댓값/최솟값|
|`Collections.reverse(list)`|List 뒤집기|

## Stream 메서드

|명령어|용도|
|---|---|
|`.distinct()`|중복 제거|
|`.filter(x -> 조건)`|조건에 맞는 것만 남김|
|`.map(x -> 변환)`|각 원소 변환|
|`.sorted()`|정렬|
|`.count()`|개수 세기|
|`.sum()`|합계 (IntStream 등)|
|`.max()` / `.min()`|최댓값/최솟값 (Optional 반환)|
|`.boxed()`|int → Integer 등 박싱|
|`.collect(Collectors.toList())`|Stream → List 변환|
|`.toArray()`|Stream → 배열 변환|
|`.anyMatch(x -> 조건)`|하나라도 만족하면 true|
|`.reduce(초기값, (a,b)->연산)`|누적 연산|

## Math

|명령어|용도|
|---|---|
|`Math.min(a, b)` / `Math.max(a, b)`|최솟값/최댓값|
|`Math.abs(x)`|절댓값|
|`Math.pow(a, b)`|거듭제곱 (결과는 double)|
|`Math.sqrt(x)`|제곱근|
|`Math.floor(x)` / `Math.ceil(x)`|내림/올림|

## String

|명령어|용도|
|---|---|
|`s.startsWith(prefix)` / `s.endsWith(suffix)`|접두/접미 확인|
|`s.substring(start, end)`|부분 문자열|
|`s.split(regex)`|문자열 분리 → String[]|
|`s.charAt(i)`|i번째 문자|
|`s.toCharArray()`|char[]로 변환|
|`String.valueOf(x)`|다른 타입 → String|
|`s.replace(a, b)`|문자열 치환|
|`s.contains(sub)`|부분 문자열 포함 여부|
|`String.join(delim, list)`|List/배열 → 하나의 String|

## Map / Set

|명령어|용도|
|---|---|
|`map.getOrDefault(k, default)`|없으면 기본값|
|`map.put(k, v)` / `map.get(k)`|저장/조회|
|`map.containsKey(k)`|key 존재 여부|
|`map.entrySet()`|key-value 쌍 순회|
|`set.add(x)` / `set.contains(x)`|추가/존재 확인 (중복 자동 제거)|
